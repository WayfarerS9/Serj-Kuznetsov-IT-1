БЛОЧНЫЕ, СТРОЧНЫЕ И БЛОЧНО-СТРОЧНЫЕ ЭЛЕМЕНТЫ БЛОК
- блочный элемент (display: block) 1. Блоки располагаются по вертикали друг под другом 2. Занимает всю строку экрана (не разрешают никому быть ни с переди ни с зади) 3. Его ширина определеяется атрибутами width, padding, border и margin; 4. Высота блочного элемента вычисляется браузером автоматически, исходя из содержимого блока, либо атрибутом height; 5. На блочные элементы не действуют свойства, предназначенные для строчных элементов 6. Блок внутри блока не выходит за пределы родителя СТРОЧНЫЙ - строчный элемент (display: inline) 1. Строчные элементы располагаются по строке вслед друг за другом. 2. Строчные элементы занимают часть строки, при этом они переносятся на другую строку при необходимости. 3. Ширина элемента равна его содержимому плюс значения margin, border и padding (но только справа и слева сверху и снизу они игнорятся). 4. Свойства, связанные с размерами (width, height), не применимы. 5. Можно выравнивать по вертикали с помощью свойства vertical-align. 6. Перенос текста считается за пробел. Строчные элементы: span, i, em, img, a, b, strong, br БЛОЧНО-СТРОЧНЫЙ 1. Несколько элементов идущих подряд располагаются на одной строке 2. Они переносятся на другую строку при необходимости. 3. Размеры содержимого можно устанавливать через свойства width и height. Ширина блока получается сложением значений width, margin, border и padding. Высота блока получается сложением значений height, margin, border и padding. 4. Высота и ширина элемента вычисляется браузером автоматически, исходя из содержимого блока. 5. Элементы можно выравнивать по вертикали с помощью свойства vertical-align. 6. Перенос текста считается за пробел. - блочно-строчный элемент (display: inline-block) - их ширина будет по контенту (не переходит на новую строку). СПИСКИ

В каждом браузере заложен размер текста, применяемый в том случае, когда этот размер явно не задан. Поэтому изначально 1em равен размеру шрифта, заданного в браузере по умолчанию или размеру шрифта родительского элемента. Следует учитывать, что размер шрифта является наследуемым элементом. Т.е. родительский элемент имеет размер своего родительского элемента. Описанное выше работает в отношении шрифтов, марджинов, хейтов (height) rem - Размер шрифта корневого элемента. привязан к корневому элементу, т. е. размеру шрифта заданного для элемента html. % - идентична em. В том смысле, что 100% = 1em. Однако размер следующих объектов учитываются следующим образом: - марджины и падинги - учитываются от ширины родительского элемента. Так, например, если родительским элементом выступает body, то марджин будет плавать при сокращении окна, т.к. ширина body, это как раз физическая ширина окна. - ширина также считается от ширины родителя; - высота считается от высоты родителя.Нужно помнить, что если нужно сделать высоту объекта до конца страницы, то нужно задать 100% высоты всем предкам вплоть до тега html. Если высоту явно не указать, то она не высчитается. - шрифт - от размера шрифта родителя. ex - Высота символа x. определяется как высота символа «x» в нижнем регистре.
